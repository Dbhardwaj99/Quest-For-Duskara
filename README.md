# Quest for Duskara

Small macOS strategy game.

Player build town.

Building make gold, food, people, and skill.

Player train soldiers.

Player sail to other towns.

Player conquer Duskara.

Simple.

## Code

- `Views/` has SwiftUI and RealityKit.
- `Views/3D/` draws town.
- `ViewModels/` connects buttons to game.
- `Models/` has game data, rules, combat, and world generation.
- `Assets/` has art and building models.
- `Tests/` checks the main game loop.

No multiplayer.

No network.

No service layer.

## Build

Open `Quest For Duskara.xcodeproj`.

Choose `Quest For Duskara`.

Build macOS app.
